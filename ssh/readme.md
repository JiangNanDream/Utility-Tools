## SSH 配置备份与恢复脚本

用于备份和恢复 Windows 用户目录下的 `.ssh` 配置（包含私钥、公钥、config 等）。

------

### 📁 目录说明

默认路径：

```powershell
%USERPROFILE%\.ssh
```

示例：

```text
C:\Users\用户名\.ssh
```

------

### 📦 功能

- 备份 `.ssh` 目录为 ZIP 文件
- 自动输出到桌面
- 恢复时自动覆盖并修复权限

------

### 🚀 使用方法

#### 1️⃣ 备份

```powershell
.\backup-ssh.ps1
```

#### 2️⃣ 恢复

```powershell
.\restore-ssh.ps1 -f "路径\ssh_backup_xxx.zip"
```

------

### ⚠️ 注意事项

- `.ssh` 目录包含**私钥（极其敏感）**
- 请勿将备份文件上传至公开平台
- 恢复操作会**覆盖现有配置**
- 恢复后已自动修复权限（避免 SSH 报错）

------

### 📌 适用场景

- 系统重装迁移 SSH 密钥
- 多设备开发环境同步
- Git / 服务器登录配置恢复
